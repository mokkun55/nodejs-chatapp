動かしたいときは<br>
.envファイルをディレクトリ直下に作成してmongoDB_URL="ここにmongoDBのURLを貼る"を記述<br>
npm iコマンドを実行 <br>
npm run devコマンドを実行<br>
→ターミナルに接続urlが表示されます<br>
